== Installation ==

1. Upload the `ai-content-engine` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to **Tools -> TonePress AI** to configure settings.
4. Enter your OpenAI API Key in the Settings tab.
5. Start generating content!

== Frequently Asked Questions ==

= Do I need an OpenAI API Key? =
Yes, you must have an active OpenAI API key to use this plugin. You can get one from the OpenAI Platform.

= Does it work with Yoast SEO? =
Yes! TonePress AI automatically fills in the Focus Keyword, Meta Title, and Meta Description fields for Yoast SEO and RankMath.

= Can I edit the articles? =
Absolutely. The plugin creates standard WordPress posts (Draft or Published) which you can edit just like any other post.
